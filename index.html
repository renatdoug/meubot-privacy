Política de Privacidade – MeuBot

Esta Política de Privacidade descreve como o MeuBot coleta, utiliza e protege os dados dos usuários durante o uso do serviço por meio da API do WhatsApp.

1. Quem somos

O MeuBot é um assistente automatizado operado por pessoa física, com finalidade de testes, estudo e demonstração de funcionalidades de automação e integração com a API do WhatsApp.

2. Dados coletados

Durante o uso do MeuBot, podem ser coletados os seguintes tipos de dados:

  • Dados de identificação básica: número de telefone utilizado na conversa.
  • Conteúdo das mensagens:
  • Dados técnicos:

3. Finalidade do uso dos dados

Os dados coletados são utilizados para:

  • Permitir o funcionamento correto do MeuBot, respondendo às mensagens enviadas pelo usuário;
  • Registrar histórico básico das interações, quando necessário, para melhoria do serviço;
  • Realizar testes, ajustes e depuração de integrações com a API do WhatsApp;
  • Cumprir obrigações legais ou solicitações de autoridades competentes, se aplicável.

4. Compartilhamento de dados

Os dados tratados pelo MeuBot não são vendidos, alugados ou compartilhados com terceiros para fins comerciais.

Os dados podem ser acessados apenas por serviços estritamente necessários ao funcionamento do bot, como:

  • A própria Meta / WhatsApp, por meio da API oficial;
  • Serviços de hospedagem e execução de código, caso o bot esteja rodando em servidores externos.

5. Base legal e consentimento

Ao iniciar uma conversa com o MeuBot e enviar mensagens voluntariamente, o usuário manifesta seu consentimento para o tratamento dos dados necessários ao funcionamento do serviço.

6. Segurança das informações

São adotadas medidas razoáveis de segurança técnica e organizacional para proteger os dados contra acessos não autorizados, perda ou alteração. No entanto, nenhum sistema é totalmente isento de riscos, e o usuário reconhece que a transmissão de dados pela internet envolve algum nível de vulnerabilidade.

7. Retenção e exclusão de dados

Os dados podem ser mantidos pelo tempo necessário para:

  • Garantir o funcionamento do MeuBot;
  • Cumprir obrigações legais ou regulatórias, se houver;
  • Realizar depuração e melhoria do serviço.

O usuário pode solicitar a exclusão dos seus dados de registro e histórico de conversa, conforme descrito na seção “Direitos do usuário”.

8. Direitos do usuário

O usuário pode, a qualquer momento:

  • Solicitar informações sobre quais dados são mantidos;
  • Solicitar correção ou atualização de dados incorretos;
  • Solicitar a exclusão de dados, quando tecnicamente possível e não conflitar com obrigações legais.

Para exercer esses direitos, basta entrar em contato pelo e-mail: renatodoug@hotmail.com.

9. Cookies e rastreamento

O MeuBot, por si só, não utiliza cookies diretamente, mas serviços externos (por exemplo, hospedagem de páginas ou dashboards) podem utilizar cookies para fins de estatística ou funcionamento técnico. Esses serviços seguem suas próprias políticas de privacidade.

10. Alterações nesta Política de Privacidade

Esta Política de Privacidade pode ser atualizada periodicamente para refletir ajustes no serviço ou exigências legais. A versão mais recente estará sempre disponível neste endereço.
